# Nickname help

{% hint style="info" %}
You are only able to adjust the colours of your name, not your displayname.
{% endhint %}

**Command Usage:** `/nick <username with colour codes>`

## Formatting your name with Colour Codes

Example:

```
&6Not&4ch
```
